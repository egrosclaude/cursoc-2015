3. Tipos de datos y expresiones


Información Adicional


Expresiones

Una expresión, en un Lenguaje de Computación,  es una secuencia de operadores y operandos que describe un cálculo. Normalmente una expresión se evalúa en tiempo de ejecución.

Ejemplos en Pascal y C:

a :real;
b :integer;
c :integer;    

float a;
int b;
int c;   

Expresiones:
                              

a :=  b * c;

a = b * c;



Overflow


Se produce lo que se conoce como Overflow cuando el dato que se desea almacenar tiene una cantidad de bits mayor que objeto destinado para su almacenamiento. Es decir que la cantidad de dígitos binarios para representar dicho número es mayor que la disponible por el objeto o porción de menmoria destinada a almacenarlo.Esto ocurre frecuentemente en operaciones con enteros.
